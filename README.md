## DL_LAB01.ipynb

This notebook demonstrates the implementation of a deep learning model for predicting student academic performance. It covers data preprocessing techniques such as handling categorical variables, feature scaling, train-test splitting, and building an Artificial Neural Network (ANN) using TensorFlow/Keras. The model is trained, evaluated, and analyzed using appropriate performance metrics.

## DL_LAB02.ipynb

This notebook focuses on garbage image classification using a Convolutional Neural Network (CNN). It includes image loading, preprocessing, data augmentation, CNN architecture design, model training, validation, and performance evaluation. The project aims to classify different categories of waste effectively using deep learning techniques.
